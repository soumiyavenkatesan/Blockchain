Setting Up MetaMask for Hardhat Network

1Install MetaMask

If you haven't already, install the MetaMask extension for your browser from metamask.io.

2Add Hardhat Network to MetaMask

Open MetaMask and follow these steps:

  1. Click on the network dropdown at the top (it might say "Ethereum Mainnet")
  2. Click "Add network"
  3. Click "Add a network manually"
  4. Fill in the following details:
    Network Name: Hardhat
    New RPC URL: http://127.0.0.1:8545
    Chain ID: 31337
    Currency Symbol: ETH
    Block Explorer URL: (leave blank)
  5. Click "Save"

3Import a Test Account

To interact with the contracts, you need to import one of the test accounts:

  1. In MetaMask, click on your account icon in the top-right corner
  2. Select "Import account"
  3. Enter the following private key:
    0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80
  4. Click "Import"

IMPORTANT: This is a test account with a publicly known private key. NEVER use this account on mainnet or any public network!

4Verify Connection

After importing the account:

  1. Make sure "Hardhat" is selected in the network dropdown
  2. You should see a balance of 10,000 ETH
  3. Return to the Yield Optimizer app and click "Connect Wallet"
  4. You should now be able to interact with the contracts
